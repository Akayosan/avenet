Ethereum L2 Network

Avenetwork is a new network that uses the L2 Etherium system so security on the network will be very safe, Avenetwork has been in development for about 3 years, now Avenetwork is here for investors and everyone to increase the security of their digital money.

Development

With Smart Contracts, every developer will be able to easily create his own project on the EXPBLOCK Smart Chain, compatible with all available tools making development open to everyone..

POA

EXPBLOCK is a Proof of Authority for the Smart Chain, the fees is distributed to validators mine Blocks on Blockchain..

Always secure

With the POA and DDOS protection, making transactions on the Blockchain is 100% safe.
